# L'Intelligence Artificielle en Cybersécurité:protection et menace

## Membres du Groupe :
* Roukaye/cheikh abdi vall(25109),toutou/demin(25161),issma/minih(25086),med lemin/el moustapha(25082).
## Description :
Ce projet explore l'impact de l'intelligence artificielle sur la cybersécurité, en analysant à la fois les opportunités de protection et les menaces potentielles. Il comprend une étude statistique basée sur un questionnaire en ligne.

## Structure du Dépôt :
* **Analyse_EXCEL** : Analyse des données et graphiques.
* **Questionnaire** : Lien vers le formulaire Google Forms.
* **Article_LaTeX** : Rédaction de l'article scientifique.
* **Presentation** : Support de présentation (Beamer).
